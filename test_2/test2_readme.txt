Задание
Даны 2 файла: золотой стандарт разметки и корпус, размеченный вручную. В корпусе 110 000 примеров, из которых 10 000 - это honeypots, взятые из файла с золотым стандартом.
Задача: сверить 10 000 примеров ручной разметки с золотым стандартом, на выходе получить файл с расхождениями (id примера, исходный текст, ручная разметка, золотая разметка).
Задача со звездочкой: найти самые частые ошибки.



fifth_dataset - корпус с разметкой 


столбец А - порядковые номера, можно игнорировать
idx - id, соответствующие id в золотом стандарте
text - исходный текст без разметки
preanot	 - автоматическая предразметка, можно игнорировать (ее не оцениваем и не сравниваем)
столбец E - разметка, которую нужно сверить с золотым стандартом


gold_dataset - золотой стандарт

столбец А - порядковые номера, можно игнорировать
Unnamed: 0 - id
source - исходный текст без разметки
annot - золотой стандарт разметки
